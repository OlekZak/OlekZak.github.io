Project Info

This project was developed as a comission for a collegue who desired a web portfolio to show-off their works which consist of 3D media.

NOTE: This project is mostly mobile optimised!

Languages used for this project:

⊙ HTML5

⊙ CSS3

⊙ JavaScript

For this project, I worked off of the clients breif, as well as mock-ups to create the site they desired. For the overall sites asthetics, I followed the mock-up quite extensively with the only deviation being the side bar. The client wanted a basic side bar, but I decided to make the sidebar fluidly animated as this gave the site much more appealing UI to navigate. After I created the base for the site, I gave the client a comprehensive guide on how to populate the site with their work, as well as how they can make everything look organised whilst not having to learn any HTML or CSS. The currently visible state of the site is post population by the client.
