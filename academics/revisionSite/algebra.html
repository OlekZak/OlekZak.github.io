Algebra

What is Algebra?

Algebra is the mathematical notation of statements that show a relation between things that vary, for example the relation between the supply, the demand and the price of an item. In these mathematical notations some items are shown with the use of letters which you may be asked to find the value of.

How to calculate Algebra?

When working with Algebra, you are essentially solvig a puzzle. When solving you must remember that what ever you do to one side you must do the same to the otherside to keep everything balanced. For Example: $$5x + 23 = -42$$ First you take away 22 from the left and right like so $$5x = -65$$ And then you divide by 5 like so $$x = -13$$

Within Algebra, you should also be able to simplify aside with ease, regardless of how many brackets there are in the equasion. Hereis a video that goes through simplifying algabraic expressions that shows how its done.

Algebra is used in all sectors of everyday life, from budgeting anf finance, to programming and even cooking, allowing you to get a better understanding of the world around you.

Quadratic Equations

What are Quardratic Equations?

Quadratic equasions are the mathematical representations fo curves, unlike linear equasions which are for lines! These equasions look like this: $$5x^2 + 4x + 2 = 0$$

How to calculate Quardratic Equations?

Calculating quadratics is the same as standard algebra, with the only caviat being that you can plot the point you get, and you usually get asked to do this between a range of values.

Another thing to note is that quadratica may not appear in their standard form, requiring you to rearrange the equasion to put them into standard form.

One thing to keep an eye out are questions asking you to complete the square, sometimes also called the vertex form. This is usually represented as: $$ax^2 + bx+ c = 0 \rightarrow a(x+d)^2 + e = 0$$ This proccess is reversable!

Finally, with either form you can find the vertex, but also the points at which the line crosses the X axis, also called the X intercept

Applications of Quardratic Equations?

Quadratic equasions, just like all other parts of Algebra can be used to calculate speed, profits but also the trajectory of anything meaning this can be used in all disciplines from graphical design to games development!

Systems of Equations

What are Systems of Equations?

Systems of Equations is the proccess of calculating variables values from 2 or more algebraic equasions with shared variables, such as x and y.

How to calculate Systems of Equations?

When calculating Systems of Equations, we calculate them at the same time as the answer of the first equasion helps with solving the second equasion. for example with this equasion: $$2x + 4y = 14$$ $$4x - 4y = 4$$ We first solve for one of the terms, any will do, for this example we'll solve for x $$2x + 4y = 14$$ $$2x = 14 - 4y$$ $$x = 7 - 2y$$ As we now know the value of x, we can use that to replace x in any other equasion like so $$4x - 4y = 4 \rightarrow 4(7 - 2y) - 4y = 4$$ $$28 - 8y - 4y = 4$$ Then we simplify like so $$28 - 12y = 4$$ We can finally solve for y $$-12y = -24$$ $$-y = -2$$ $$y = 2$$ From here we solve for x with the value of y $$2x + 4y = 14 \rightarrow 2x + 8 = 14$$ $$2x = 6$$ $$x = 3$$ With all of this inforamation we know the results of all the variables in this equasion $$x = 3 \:\:\:\: y = 2$$ This can be done with multiple equasions as long as there are less or equal variables to the number of equasions.
